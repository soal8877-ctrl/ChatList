# Changelog

Все значимые изменения в проекте ChatList будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2024-01-XX

### Добавлено
- Первый релиз приложения ChatList
- Поддержка множественных AI моделей (OpenAI, DeepSeek, Groq, OpenRouter)
- Графический интерфейс на PyQt5
- Сохранение промтов с тегами
- Сохранение результатов в базу данных SQLite
- Просмотр результатов в табличном формате и Markdown
- Экспорт результатов в Markdown и JSON
- Управление моделями (добавление, редактирование, удаление, активация/деактивация)
- Детальное логирование всех запросов
- Настройки таймаутов и темы оформления
- Инсталлятор для Windows с помощью Inno Setup
- Автоматическая сборка и публикация через GitHub Actions
- GitHub Pages сайт с лендингом

### Исправлено
- Исправлена проблема с правами доступа к базе данных при установке в Program Files
- Исправлена проблема с созданием директории логов при установке
- База данных и логи теперь сохраняются в пользовательской директории AppData

### Технические детали
- Python 3.7+
- PyQt5 для GUI
- SQLite для хранения данных
- PyInstaller для создания исполняемого файла
- Inno Setup для создания инсталлятора
