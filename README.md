# Минимальное PyQt приложение

Простое приложение на Python с графическим интерфейсом на PyQt5.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск приложения

```bash
python main.py
```

## Описание

Минимальное приложение на PyQt5 с графическим интерфейсом:
- Окно с заголовком
- Метка с текстом
- Кнопка, которая изменяет текст метки при нажатии
