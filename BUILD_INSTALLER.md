# Инструкция по созданию инсталлятора ChatList

## Способ 1: Автоматическая сборка (рекомендуется)

1. **Убедитесь, что установлен Inno Setup:**
   - Скачайте с https://jrsoftware.org/isdl.php
   - Установите Inno Setup (обычно в `C:\Program Files (x86)\Inno Setup 6\`)

2. **Соберите исполняемый файл приложения:**
   ```bash
   python build.py
   ```

3. **Создайте инсталлятор:**
   ```bash
   python build_installer.py
   ```

   Инсталлятор будет создан в папке `dist` как `ChatList-Setup-1.0.0.exe`

## Способ 2: Ручная сборка через Inno Setup Compiler

1. **Установите Inno Setup** (если еще не установлен):
   - Скачайте с https://jrsoftware.org/isdl.php
   - Установите Inno Setup

2. **Соберите исполняемый файл приложения:**
   ```bash
   python build.py
   ```

3. **Откройте Inno Setup Compiler:**
   - Запустите `Inno Setup Compiler` из меню Пуск

4. **Откройте скрипт:**
   - File → Open
   - Выберите файл `installer.iss`

5. **Скомпилируйте инсталлятор:**
   - Build → Compile (или нажмите F9)
   - Инсталлятор будет создан в папке `dist`

## Способ 3: Командная строка

Если Inno Setup установлен, можно использовать командную строку:

```bash
# Сначала соберите приложение
python build.py

# Затем создайте инсталлятор
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

## Использование инсталлятора

После создания инсталлятора:

1. **Запустите** `dist\ChatList-Setup-1.0.0.exe`
2. Следуйте инструкциям мастера установки
3. После установки программа будет доступна в меню Пуск и на рабочем столе (если выбрано)

## Деинсталляция

Программу можно удалить через:
- Панель управления → Программы и компоненты → ChatList 1.0.0 → Удалить
- Или через меню Пуск → ChatList → Uninstall ChatList
